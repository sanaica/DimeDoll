# DimeDoll

**Calm AI-Powered Financial Wellness & Trading Simulator**

DimeDoll replaces intimidating finance culture with a peaceful, welcoming experience. Users define their risk profile, set goals, and use our AI Auto-Pilot + risk-free simulator to build investing confidence.

## Features
- User profile with risk assessment & capital tracking
- Gemini-powered personalized investment recommendations
- Real-time market simulation engine
- Full-stack architecture (React + FastAPI + Docker)

## Tech Stack
**Frontend**: React, React Router, Recharts, TailwindCSS  
**Backend**: FastAPI, Python, Gemini Flash, yfinance  
**Database**: Supabase / SQLite (for dev)  
**Others**: Docker, Docker Compose
